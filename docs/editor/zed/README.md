# Zed editor configuration (backup)

Mirrors the local Zed user config from `~/.config/zed/` on this machine (macOS). Zed stores JSON with comments (JSONC); trailing commas are valid there.

| File in repo        | Install location                    |
| ------------------- | ----------------------------------- |
| `settings.json`     | `~/.config/zed/settings.json`       |
| `keymap.json`       | `~/.config/zed/keymap.json`         |

Upstream docs: [Configuring Zed](https://zed.dev/docs/configuring-zed), [Key bindings](https://zed.dev/docs/key-bindings).
